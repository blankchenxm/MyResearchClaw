ThingML: a language and code generation framework for heterogeneous targets
★ 0
Nicolas Harrand et al.
MoDELS 2016 model-driven code-generation heterogeneous-targets
EN
ThingML provides a model-driven language and code generation framework for heterogeneous targets, with customizable generators that can emit code for multiple runtime environments. It is directly relevant because it shows how a single abstraction can be retargeted across embedded platforms instead of hand-maintaining separate firmware code paths for each MCU family.
中文

Generates platform-specific code from platform-independent timed models, showing how model-based synthesis can preserve timing constraints while targeting embedded deployments. RapidPatch: Firmware Hotpatching for Real-Time Embedded Devices
★ 163
Yi He et al.
USENIX Security 2022 hotpatching real-time firmware
EN
Introduces firmware hotpatching for real-time embedded devices, aiming to update code without breaking timing guarantees or rebooting the system. The technical significance is that it moves embedded firmware maintenance from full reflash cycles toward surgical patch injection, which is closer to how production fleets need to be serviced.
中文
该工作提出面向实时嵌入式设备的固件热补丁，目标是在不破坏时序约束、也不重启系统的前提下更新代码。其技术意义在于把嵌入式固件维护从整机重刷推进到更细粒度的补丁注入，这更接近量产设备车队的实际运维需求。
