PCBSchemaGen: Constraint-Guided Schematic Design via LLM for Printed Circuit Boards (PCB)
★ 0
Huanghaohe Zou et al.
CoRR 2026 pcb-schematic llm constraint-guided
EN
PCBSchemaGen proposes a training-free PCB schematic design pipeline built from an LLM agent and constraint-guided synthesis. It adds a verification layer based on a real datasheet-derived knowledge graph and subgraph isomorphism so that pin roles and topological constraints are checked before a schematic is accepted. The paper evaluates 23 schematic tasks across digital, analog, and power domains, which makes it unusually close to a practical requirement-to-schematic generator. For this topic, it is the clearest bridge from a high-level board brief to a structured design artifact.
中文
PCBSchemaGen 提出了一条无需训练的 PCB 原理图设计流水线，由 LLM agent 和约束引导的综合模块组成。它额外加入了基于真实 datasheet 构建的知识图谱与子图同构校验，用来在接受原理图之前检查引脚角色和拓扑约束。论文在数字、模拟和电源三类任务上共评估了 23 个 PCB schematic 设计场景，因此非常接近真正的“需求到原理图”生成器。对这个主题来说，它是从高层板级需求落到结构化设计工件最直接的一篇。

OmniSch: A Multimodal PCB Schematic Benchmark For Structured Diagram Visual Reasoning
★ 0
Taiting Lu et al.
CoRR 2026 benchmark pcb-schematic multimodal
EN
OmniSch turns PCB schematic understanding into a benchmark problem for multimodal models and agentic search. The benchmark contains 1,854 real-world schematic diagrams and tests four capabilities: entity grounding, diagram-to-graph reasoning, geometric reasoning, and tool-augmented visual search. That framing matters because PCB automation breaks down quickly if a model cannot read an existing schematic and recover connectivity plus spatial constraints. For requirement-to-PCB generation, OmniSch is the missing evaluation layer between raw vision models and design automation.
中文
OmniSch 把 PCB 原理图理解问题抽象成一个面向多模态模型的基准任务。它包含 1,854 张真实原理图，并考察四类能力：实体定位、图结构推理、几何推理以及工具增强的视觉检索。这个设定很重要，因为如果模型连已有原理图里的连线和空间约束都读不准，PCB 自动化流程就会很快失效。对于需求到 PCB 的生成路径来说，OmniSch 提供了连接视觉模型与设计自动化之间缺失的评测层。

PCBSmith: An Effective Schematic Generator for Testing PCB Design Tool Chain
★ 2
Xu Zhao et al.
IEEE Trans. Reliability 2025 schematic-generation eda tool-chain
EN
PCBSmith is a schematic generator designed specifically to stress-test a PCB design tool chain. It mimics the steps a PCB designer would take by selecting components, assigning electrical parameters, and assembling a simulatable schematic rather than just emitting a symbolic sketch. That makes it useful as both a generator and a validation workload for downstream EDA automation. In this topic, it is the most explicit example of turning a board-level design intent into executable schematic artifacts.
中文
PCBSmith 是一个专门用来测试 PCB 设计工具链的原理图生成器。它模拟 PCB 设计者的步骤：先选器件，再设置电气参数，最后拼装出可以仿真的原理图，而不是只输出一个符号化草图。这样它既是生成器，也是下游 EDA 自动化链路的验证负载。放在这个主题下，它是把板级设计意图转成可执行 schematic 工件最明确的例子之一。

PCBAgent: An Agent-based Framework for High-Density Printed Circuit Board Placement
★ 3
Lin Chen et al.
ASP-DAC 2025 placement rl llm
EN
PCBAgent addresses the placement side of PCB automation with an agent-based framework for high-density boards. The design combines an RL agent for layout inference and fine-tuning with an LLM agent for interactive optimization, which is a strong fit for constrained board layouts where human feedback still matters. The paper reports experiments on 17 industrial tasks and says it outperforms prior methods. For IoT sensor boards, this is the part of the pipeline that turns a schematic into something that can actually fit on the board outline.
中文

HiePlace: Efficient Hierarchical PCB Placement
★ 0
Shanyi Li et al.
IEEE TCAD 2025 placement mathematical-programming optimization
EN
HiePlace is a mathematical-programming-based placement framework tailored to PCB constraints rather than chip placement. The paper introduces a dynamic-programming stage for core components and a relaxation method that shrinks the boolean search space, which together improve runtime and solution quality. The available abstract summary reports a 7.7x speedup and a 66% cost reduction. That combination is exactly what board designers need when a dense IoT sensor stack must satisfy clearance, alignment, and mechanical constraints at the same time.
中文
HiePlace 是一个面向 PCB 约束而不是芯片约束的数学规划式 placement 框架。它引入动态规划步骤来优先放置核心器件，并用 relaxation 方法缩小布尔变量搜索空间，从而同时提升运行时间和布局质量。现有摘要信息给出的结果是 7.7x 的加速和 66% 的成本下降。对于需要同时满足间距、对齐和机械约束的高密度 IoT 板来说，这正是非常实用的优化形态。

PCBRouteNet: Dynamic Quadrilateral-Flow Dataset and Benchmarks for Machine-Learning PCB Routing
★ 0
Zhihao Ren et al.
IEEE TVLSI 2025 routing dataset benchmark
EN
PCBRouteNet focuses on routing, but its main contribution is the dataset and benchmark setup that make ML routing work practical. It converts PCB layouts into a quadrilateral network-flow representation so that global and detailed routing tasks can be learned from large-scale board data. The abstract summary also emphasizes multilayer boards, high-density interconnects, and scaling-law style analysis of the flow encoding. This is the infrastructure paper you want if the goal is to learn routing policies instead of hand-coding heuristics for every new board class.
中文
PCBRouteNet 的重点不是单一 router，而是为机器学习 PCB routing 提供可训练的数据集与基准。它把 PCB 布局转换成四边形网络流表示，使全局 routing 和细节 routing 都能在大规模板级数据上学习。摘要还特别强调了多层板、高密度互连以及对 flow 编码的 scaling-law 分析。若目标是训练 routing policy，而不是每种新板型都手写启发式规则，这篇就是基础设施型工作。

EEschematic: Multimodal-LLM Based AI Agent for Schematic Generation of Analog Circuit
★ 0
Chang Liu et al.
CoRR 2025 analog-schematic multimodal-llm arxiv
EN
EEschematic converts SPICE netlists into human-editable schematic diagrams with a multimodal LLM agent. It uses a small set of analog substructure examples for few-shot placement and a Visual Chain-of-Thought loop to refine placement and wiring until the drawing becomes clearer and more symmetric. The evaluated circuits are classic analog building blocks such as an inverter, 5T-OTA, and telescopic cascode amplifier. It is not PCB-specific, but it is relevant because sensor boards often include analog front ends that need the same schematic-generation discipline.
中文
EEschematic 把 SPICE netlist 转成可人工编辑的 schematic diagram，并且用的是 multimodal LLM agent。它用少量 analog substructure 示例做 few-shot placement，再通过 Visual Chain-of-Thought 循环不断修正器件摆放和连线，让最终图更清晰、更对称。评估对象是 inverter、5T-OTA、telescopic cascode amplifier 这类经典模拟电路。它不是 PCB 专属工作，但对传感器板里的模拟前端很相关，因为这类模块同样需要稳定的原理图生成能力。
